Assessment & Suggestions

📘 About

This calculator estimates the overflow velocity (surface loading), a dimensionless friction factor, and resulting head loss for flow through and over a porous/settling medium using Carman–Kozeny style relations. Provide inputs using the units shown for consistent results.

✅ Input Validations

  • Surface Area (A): must be > 0 m². Block if <0.01 m². Warn if <0.1 or >1,000,000 m².
  • Flow Rate (Q): must be > 0 m³/day. Block if <1e-4 or >1e9. Warn if <0.01 or >1e8.
  • Porosity (n): must be between 0 and 0.95. Warn if <0.2 or >0.7.
  • Modified Reynolds Number (Ng): must be ≥ 1e-6. Warn if <1e-3 or >1e6.
  • Particle Diameter (dg): must be between 1e-7 m and 5e-2 m. Warn if <1e-6 m or >1e-2 m.
  • Shape Factor (Φ): must be between 0 and 5. Warn if <0.4 or >1.5.
  • Overflow Velocity (vs): blocked if >1000 m/day. Warn if >100 or <1 m/day.
  • Head Loss (h): warnings if >1 m (high), critical warning if >10 m.
  • All inputs must be finite numbers (no blanks, text, NaN, or infinity).
  • Division-by-zero and denominator stability checks enforced (Φ·n³·dg ≥ 1e-12).

🌍 Applications

  • Municipal Water Treatment: Clarifier/settler performance checks and sizing.
  • Industrial Effluent Management: Designing sedimentation basins and solids removal systems.
  • Stormwater Control: Sizing detention basins and sedimentation ponds.
  • Hydraulic & Environmental Engineering: Evaluating energy loss and flow resistance in porous media.
