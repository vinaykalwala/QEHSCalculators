📘 About

This calculator applies a correction to an assumed flow to obtain the corrected flow value. Inputs and outputs are in m³/s. Logs should be maintained for audits and safety reviews.

⚠️Input Validations

  • ✔️ Assumed Flow must be a positive value (> 0)
  • ✔️ Flow Correction can be positive or negative (numeric only)
  • ❌ Corrected Flow ≤ 0 → invalid (flow reversal)
  • ❌ Values outside 1e-9 – 1e6 m³/s → rejected (likely unit error)
  • ⚠️ |ΔQ| > 10× Qassumed → warning (check inputs)
  • ⚠️ Too many decimals (> 6) → rounded

🌍 Applications

  • Loop Balancing: Hardy-Cross corrections in water networks
  • Process Safety: Correcting assumed flows for firewater systems
  • Audits: Documenting changes in assumed vs corrected values
