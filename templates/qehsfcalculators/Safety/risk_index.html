📘 About This Calculator

The Risk Index Calculator computes net risk by comparing hazards against existing controls. A positive result means adequate controls, while a negative result indicates residual risk requiring attention.

⚙ Parameters Used

  • Control Score: Effectiveness of existing controls (higher = safer)
  • Hazard Score: Severity × Probability of hazard (higher = riskier)

✅ Input Validation Rules

  • Control Score: 0–100, decimals allowed (step 0.1), no negatives
  • Hazard Score: 0–100, decimals allowed (step 0.1), no negatives
  • Both must use the same scoring scale (0–10 or 0–100)
  • Blank or non-numeric entries are invalid
  • Cross-check: Control = 0 & Hazard > 0 → always dangerous

📊 Output Validations

Formula: Risk Index = Control Score – Hazard Score

  • RI > 0 → ✅ SAFE: Controls adequately address hazards.
  • RI = 0 → ⚠️ BORDERLINE: Controls just match hazards – consider improvements.
  • -10 < RI < 0 → ⚠️ UNSAFE: Hazards exceed controls – needs attention.
  • RI ≤ -10 → ❌ DANGEROUS: Significant uncontrolled hazards – immediate action required!

📏 Standards

This approach aligns with ISO 31000 (Risk Management), OSHA Hazard Ranking, and NFPA Fire Safety Risk Methods. It provides a transparent metric for safety audits, incident prevention, and compliance reporting.

📝 Conclusion

The Risk Index Calculator provides a clear and transparent way to measure whether existing controls adequately offset hazards.

This approach aligns with ISO 31000, OSHA, and NFPA risk assessment principles, making it a practical decision-support tool for incident prevention, compliance reporting, and continuous improvement in workplace safety.
