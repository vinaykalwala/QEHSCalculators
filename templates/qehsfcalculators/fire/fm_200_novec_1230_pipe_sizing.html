📘 About This Calculator

This calculator determines the optimal pipe diameter for FM-200 / NOVEC 1230 fire suppression systems, ensuring proper agent distribution, reliable system performance, and compliance with safety standards.

⚙️ Parameters Used

  • Flow Rate (Q): Mass of suppression agent released per second (kg/s).
  • Velocity (V): Recommended agent transport speed in the pipe (m/s).
  • Agent Type: FM-200 or NOVEC 1230, each with its own density (FM-200 ~1400 kg/m³, NOVEC 1230 ~1600 kg/m³).

These parameters are used to calculate the minimum required internal diameter of the suppression piping, expressed in centimeters.

✅ Input Validation Rules

  • All input fields are mandatory and must be numeric.
  • Flow Rate (kg/s):
    • Must be > 0.1 kg/s (practical minimum).
    • Values above 200 kg/s trigger a caution warning.
    • Decimals up to 2 places allowed.
  • Velocity (m/s):
    • Minimum: 2 m/s (below this, agent transport is ineffective).
    • Maximum: 30 m/s (NFPA 2001 safe limit).
    • Decimals up to 2 places allowed.
  • Agent Type: Must be selected (FM-200 or NOVEC 1230).

📏 Output Validations

  • Pipe Diameter (cm):
    • < 2 cm → ⚠️ Too small, unsafe for real systems.
    • 2–30 cm → ✅ Valid range for fire suppression piping.
    • > 30 cm → ⚠️ Unrealistic, recheck input values.

The calculator displays these warnings or confirmations directly with the result, helping users verify practicality and safety.

📐 Governing Principle

An FM-200 / NOVEC 1230 pipe sizing calculator follows the standards set by NFPA 2001, the American standard, and can also adhere to ISO 14520 or the Indian standard IS 15493, which are international and Indian equivalents, respectively. These standards ensure proper agent distribution, prevent pressure drops, and maintain required flow rates for effective fire suppression.

📘 Standards for FM-200 / NOVEC 1230 Systems
  • NFPA 2001: The primary standard for clean agent fire extinguishing systems, covering guidelines for designing, installing, and maintaining these systems.
  • ISO 14520: An international standard for gaseous fire extinguishing systems, providing equivalent requirements to NFPA 2001.
  • IS 15493: The Indian standard for gaseous fire extinguishing systems, outlining general requirements for systems using FM-200 or NOVEC 1230.
⚖️ Importance of Pipe Sizing
  • Even Agent Distribution: Correctly sized pipes ensure the agent is discharged evenly to all protected areas.
  • Regulatory Compliance: Adhering to these standards is crucial for meeting safety and performance regulations.
  • System Protection: Proper pipe sizing prevents over-pressurization, which can damage the system and pipes.

📌 Conclusion

This tool supports engineers, fire safety professionals, and system designers in validating pipe diameters for FM-200 and NOVEC 1230 systems. By integrating industry-standard validation checks, it ensures that results are practical, safe, and compliant with NFPA 2001 and ISO 14520 requirements.
